import { Accordion, AccordionItem, AccordionTrigger, AccordionContent } from "yunui";

A Radix-backed, vertically stacked set of collapsible sections. Set `type="single"` for one-at-a-time panels (add `collapsible` so the open one can close) or `type="multiple"` to allow several open at once. Full keyboard a11y (arrow keys, Home/End, Enter/Space) comes from Radix.

## Import

```tsx
import {
  Accordion, AccordionItem, AccordionTrigger, AccordionContent,
} from "@yuhuanowo/yunui";
```

## Props

`Accordion`, `AccordionItem`, `AccordionTrigger` and `AccordionContent` are thin wrappers over Radix's Accordion primitives, so they accept all of their props. The most common ones:

<PropsTable
  rows={[
    { name: "type", type: '"single" | "multiple"', required: true, description: "Whether one or several panels can be open at once." },
    { name: "collapsible", type: "boolean", defaultValue: "false", description: "(type=single) Allow the open panel to be closed." },
    { name: "value / defaultValue", type: "string | string[]", description: "Controlled / uncontrolled open item(s)." },
    { name: "onValueChange", type: "(value) => void", description: "Fires when the open item(s) change." },
    { name: "AccordionItem.value", type: "string", required: true, description: "Unique identifier for the section." },
    { name: "disabled", type: "boolean", description: "Disable an item or the whole group." },
  ]}
