import { Accordion, AccordionItem, AccordionTrigger, AccordionContent } from "yunui";

一個基於 Radix、垂直堆疊的可折疊區塊集合。設定 `type="single"` 可一次只展開一個面板（再加 `collapsible` 讓已展開的面板可以收起），或設定 `type="multiple"` 允許同時展開多個。完整的鍵盤無障礙支援（方向鍵、Home/End、Enter/Space）來自 Radix。

## 匯入

```tsx
import {
  Accordion, AccordionItem, AccordionTrigger, AccordionContent,
} from "@yuhuanowo/yunui";
```

## 屬性

`Accordion`、`AccordionItem`、`AccordionTrigger` 和 `AccordionContent` 都是對 Radix Accordion 原語的輕量包裝，因此接受它們的全部屬性。最常用的幾個：

<PropsTable
  rows={[
    { name: "type", type: '"single" | "multiple"', required: true, description: "一次可展開一個還是多個面板。" },
    { name: "collapsible", type: "boolean", defaultValue: "false", description: "（type=single）允許關閉已展開的面板。" },
    { name: "value / defaultValue", type: "string | string[]", description: "受控 / 非受控的展開項。" },
    { name: "onValueChange", type: "(value) => void", description: "展開項變化時觸發。" },
    { name: "AccordionItem.value", type: "string", required: true, description: "該區塊的唯一識別碼。" },
    { name: "disabled", type: "boolean", description: "停用單個項或整個組。" },
  ]}
/>
