import { Table, TableHeader, TableBody, TableFooter, TableRow, TableHead, TableCell, TableCaption, Badge } from "yunui";

一个语义化、纯展示型、基于设计系统令牌构建的表格组件族。`Table` 会把 `<table>` 包裹在一个溢出容器中，使宽表格滚动而不破坏布局。用 `TableHeader`、`TableBody`、`TableRow`、`TableHead` 和 `TableCell` 来组合它（短别名 `Thead`、`Tbody`、`Tr`、`Th`、`Td` 也都有导出）。

## 导入

```tsx
import {
  Table, TableHeader, TableBody, TableFooter,
  TableRow, TableHead, TableCell, TableCaption,
} from "@yuhuanowo/yunui";
```
