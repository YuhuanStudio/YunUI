import { Select, SelectTrigger, SelectValue, SelectContent, SelectItem } from "yunui";

基於 Radix `Select` 建構的下拉選擇框——由一系列組合部件構成，渲染出一個掛載到 portal、帶動畫、支援鍵盤存取的列表框。如需扁平的、基於 options 陣列的 API（帶搜尋），請參閱 [Combobox](/docs/components/combobox)；如需完全自訂樣式的變體，請參閱 [CustomSelect](/docs/components/custom-select)。

## Import

```tsx
import {
  Select,
  SelectTrigger,
  SelectValue,
  SelectContent,
  SelectItem,
} from "@yuhuanowo/yunui";
```

## Parts

| 部件 | 作用 |
| --- | --- |
| `Select` | 根元件；持有 `value`/`onValueChange` 或 `defaultValue`。 |
| `SelectTrigger` | 開啟列表框的按鈕。 |
| `SelectValue` | 渲染已選取的標籤（或 `placeholder`）。 |
| `SelectContent` | 掛載到 portal、帶動畫的下拉面板。 |
| `SelectItem` | 單個選項（`value` 為必填）。 |

`Select` 重新匯出了帶有設計系統樣式的 Radix `Select` 部件。完整的屬性參考（包括 `SelectGroup`）請參閱 [Radix Select 文件](https://www.radix-ui.com/primitives/docs/components/select)。
