import { Textarea } from "yunui";

多行文字輸入框。與 `Input` 使用相同的樣式語言，內聯的 `error` 錯誤訊息同樣會將邊框切換為錯誤樣式。所有原生 `<textarea>` 屬性都會透傳。

## 匯入

```tsx
import { Textarea } from "@yuhuanowo/yunui";
```
