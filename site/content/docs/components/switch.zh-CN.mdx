import { Switch } from "yunui";

受控的开/关切换开关。它仅支持受控模式：传入 `checked` 和 `onCheckedChange`。支持两种尺寸（`sm`、`md`）以及选中状态的四种颜色变体。

## Import

```tsx
import { Switch } from "@yuhuanowo/yunui";
```

## Usage

`Switch` 是受控组件——将它与 state 关联：

```tsx
const [on, setOn] = useState(false);
<Switch checked={on} onCheckedChange={setOn} />
```
