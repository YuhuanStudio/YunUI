import { Textarea } from "yunui";

多行文本输入框。与 `Input` 使用相同的样式语言，内联的 `error` 错误信息同样会将边框切换为错误样式。所有原生 `<textarea>` 属性都会透传。

## Import

```tsx
import { Textarea } from "@yuhuanowo/yunui";
```
