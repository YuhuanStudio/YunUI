import { Badge } from "yunui";

用於狀態和標籤的小型行內膠囊。它渲染語義化的顏色變體，並轉發所有原生 `<span>` 屬性。

## 匯入

```tsx
import { Badge } from "@yuhuanowo/yunui";
```

## 變體

## 情境中

徽章可以自然地置於文字旁，一目了然地傳達狀態。

<ComponentPreview
  title="Inline status"
  code={`<span className="inline-flex items-center gap-2 text-sm">
  Claude Opus 4.8
  <Badge variant="success">Active</Badge>
</span>`}
>

    Claude Opus 4.8
    <Badge variant="success">Active</Badge>
  </span>
</ComponentPreview>
