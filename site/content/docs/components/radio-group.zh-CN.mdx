import { RadioGroup, RadioGroupItem } from "yunui";

一个基于 Radix 的单选项集合。通过 `value`/`onValueChange` 受控（或通过 `defaultValue` 非受控）。方向键可在组内移动并选择，整个组是单个 tab 停靠点。每一项渲染一个圆环，选中时填充主强调色并显示一个中心圆点——与 YunUI 的 Checkbox/Switch 视觉语言一致。

## 导入

```tsx
import { RadioGroup, RadioGroupItem } from "@yuhuanowo/yunui";
```

## 属性

`RadioGroup` 和 `RadioGroupItem` 包装了 Radix 的 RadioGroup 原语，接受它们的全部属性。最常用的几个：
