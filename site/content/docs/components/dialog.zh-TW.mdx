import { Dialog, DialogTrigger, DialogContent, DialogHeader, DialogTitle, DialogDescription, DialogFooter, DialogClose, Button } from "yunui";

基於 Radix `Dialog` 構建的模態對話框——由一組部件組合而成，在變暗、模糊的背景之上渲染一個傳送到 body、可聚焦鎖定、支援鍵盤操作的面板。`DialogContent` 已內建遮罩層和關閉按鈕。透過 `DialogTrigger` 觸發，或使用 `open`/`onOpenChange` 自行控制。

## Import

```tsx
import {
  Dialog,
  DialogTrigger,
  DialogContent,
  DialogHeader,
  DialogTitle,
  DialogDescription,
  DialogFooter,
  DialogClose,
} from "@yuhuanowo/yunui";
```

## Parts

| 部件 | 作用 |
| --- | --- |
| `Dialog` | 根節點；持有 `open`/`onOpenChange`（或以非受控方式運行）。 |
| `DialogTrigger` | 開啟對話框的元素（用 `asChild` 包裹一個 `Button`）。 |
| `DialogContent` | 傳送、置中的面板——已內建遮罩層和關閉按鈕。 |
| `DialogHeader` | 標題與描述的佈局區塊。 |
| `DialogTitle` | 為螢幕報讀器標註對話框的無障礙標題。 |
| `DialogDescription` | 標題下方的補充文字。 |
| `DialogFooter` | 操作按鈕的靠右對齊列。 |
| `DialogClose` | 啟用後關閉對話框的元素。 |

`Dialog` 以設計系統樣式重新匯出 Radix `Dialog` 部件。完整的屬性參考（包括 `DialogPortal` 和 `DialogOverlay`）請見 [Radix Dialog 文件](https://www.radix-ui.com/primitives/docs/components/dialog)。
