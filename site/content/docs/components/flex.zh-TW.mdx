import { Flex, Card } from "yunui";

一個 flexbox 佈局基礎元件。`direction`、`align`、`justify`、`gap`、`wrap` 以及內邊距屬性會透過靜態查找表對應到字面的 Tailwind 類別，因此編譯器總會產生它們。純展示型——額外的屬性會展開到元素上，`as` 可改變標籤。

## 匯入

```tsx
import { Flex, Row, Column, Stack } from "@yuhuanowo/yunui";
```

`Row`、`Column` 和 `Stack` 是幾個輕量預設：`Row` 固定 `direction="row"`，`Column`/`Stack` 固定 `direction="col"`（`Stack` 額外帶一個預設 `gap`）。
