import { SegmentedBar } from "yunui";

將單條水平條按比例拆分為多個彩色分段——記憶體分配（活躍 / 快取 / 空閒）、按桶劃分的磁碟占用、請求構成。與 [MetricBar](/docs/patterns/metric-bar)（單一填充）不同，它把多個分段堆疊到 `total`，剩餘部分顯示為軌道。開啟 `legend` 可列出每個分段及其數值。

## 匯入

```tsx
import { SegmentedBar } from "@yuhuanowo/yunui";
```

## 基礎

傳入分段；長條會按各分段之和的比例填充。

<ComponentPreview
  title="Segments"
  code={`<SegmentedBar
  className="w-72"
  segments={[
    { value: 42, tone: "accent", label: "Active" },
    { value: 18, tone: "info", label: "Cache" },
    { value: 9, tone: "warning", label: "Peak" },
  ]}
/>`}
>
  <SegmentedBar
    className="w-72"
    segments={[
      { value: 42, tone: "accent", label: "Active" },
      { value: 18, tone: "info", label: "Cache" },
      { value: 9, tone: "warning", label: "Peak" },
    ]}
  />
</ComponentPreview>

## 總量與剩餘

傳入大於分段之和的 `total`，可將空閒空間顯示為軌道——非常適合「已用 X / 共 Y」。

<ComponentPreview
  title="Legend"
  code={`<SegmentedBar
  className="w-72"
  total={64}
  legend
  formatValue={(v) => v + " GB"}
  segments={[
    { value: 22, tone: "accent", label: "Active" },
    { value: 12, tone: "info", label: "Cache" },
  ]}
/>`}
