import { Sparkline } from "yunui";

用於單條數值序列的迷你行內折線（或面積）圖——請求吞吐、GPU 使用率、捲動視窗內的延遲。純 SVG，不依賴圖表庫。它會把資料縮放到合適範圍並撐滿容器寬度，因此能自然嵌入統計卡片或表格儲存格。`tone` 從設計權杖中選取描邊顏色，`color` 可覆蓋它。

## 匯入

```tsx
import { Sparkline } from "@yuhuanowo/yunui";
```

## 面積填充

設定 `area` 可在折線下方繪製描邊色的垂直漸層。

  title="面積"

</ComponentPreview>

## 語義色

`tone` 對應到語義顏色角色（`accent`、`success`、`warning`、`error`、`info`、`neutral`）。
