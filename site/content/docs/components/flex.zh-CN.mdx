import { Flex, Card } from "yunui";

一个 flexbox 布局基础组件。`direction`、`align`、`justify`、`gap`、`wrap` 以及内边距属性会通过静态查找表映射到字面的 Tailwind 类，因此编译器总会生成它们。纯展示型——额外的属性会展开到元素上，`as` 可改变标签。

## 导入

```tsx
import { Flex, Row, Column, Stack } from "@yuhuanowo/yunui";
```

`Row`、`Column` 和 `Stack` 是几个轻量预设：`Row` 固定 `direction="row"`，`Column`/`Stack` 固定 `direction="col"`（`Stack` 额外带一个默认 `gap`）。
