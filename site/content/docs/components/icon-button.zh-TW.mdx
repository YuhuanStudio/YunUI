import { IconButton } from "yunui";
import { Heart, Settings, Trash2 } from "lucide-react";

一個帶有內建工具提示的純圖示按鈕。必填的 `label` 同時充當無障礙名稱和工具提示文字，因此純圖示操作對螢幕閱讀器和鍵盤使用者依然可用。所有原生 `<button>` 屬性（如 `onClick` 和 `disabled`）都會透傳。

## 匯入

```tsx
import { IconButton } from "@yuhuanowo/yunui";
```
