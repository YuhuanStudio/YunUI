import { Button, IconButton } from "yunui";
import { Heart, Settings } from "lucide-react";

主操作按鈕。它渲染 YunUI 的 `.btn` 類別，支援樣式變體和尺寸，可顯示載入指示器，並能透過 `asChild` 把自身樣式套用到子元素上（例如路由用的 `<Link>`）。

## 匯入

```tsx
import { Button } from "@yuhuanowo/yunui";
```

## 圖示按鈕

對於僅含圖示的操作，請使用 `IconButton`，它會附帶一個無障礙標籤和內建提示框。

<ComponentPreview
  title="IconButton"
  code={`import { IconButton } from "@yuhuanowo/yunui";
import { Heart, Settings } from "lucide-react";

<IconButton icon={<Heart className="w-4 h-4" />} label="Like" />
<IconButton icon={<Settings className="w-4 h-4" />} label="Settings" />`}
>
  <IconButton icon={<Heart className="w-4 h-4" />} label="Like" />
  <IconButton icon={<Settings className="w-4 h-4" />} label="Settings" />
</ComponentPreview>
