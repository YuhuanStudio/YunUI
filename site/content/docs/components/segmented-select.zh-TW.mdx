import { SegmentedSelect } from "yunui";

一個內嵌的分段控制項 —— 一排按鈕，任何時刻只有一個選項處於啟用狀態。適用於簡短、互斥的選擇，例如檢視切換器或篩選模式。它是受控的：傳入 `value`、`onChange` 以及一個 `options` 陣列。每個選項都可以攜帶一個可選的 Lucide `icon` 圖示與一個原生提示 `desc`。

## Import

```tsx
import { SegmentedSelect } from "@yuhuanowo/yunui";
```

## Usage

`SegmentedSelect` 是受控元件 —— 將 `value` 與 `onChange` 與狀態繫結：

```tsx
import { LayoutGrid, List, Table } from "lucide-react";

const [value, setValue] = useState("grid");

<SegmentedSelect
  value={value}
  onChange={setValue}
  options={[
    { value: "grid", label: "Grid", icon: LayoutGrid },
    { value: "list", label: "List", icon: List },
    { value: "table", label: "Table", icon: Table },
  ]}
/>
```
