import { ConfirmModal, Button } from "yunui";

針對破壞性或不可逆操作的精簡確認對話框。它會顯示一個按變體著色的圖示、`title`、`message` 以及取消 / 確認按鈕，並支援 `isLoading` 狀態——在操作執行期間停用按鈕並顯示載入指示器。與 [Modal](/docs/components/modal) 一樣，它透過 `isOpen` + `onClose` 受控，外加一個 `onConfirm` 回呼。此外還匯出了便捷預設 `DeleteConfirmModal` 和 `RegenerateConfirmModal`。

## Import

```tsx
import { ConfirmModal } from "@yuhuanowo/yunui";
```

## Basic

將 `isOpen` 保存在 state 中，在 `onConfirm` 裡執行你的操作，並透過 `onClose` 關閉。`variant`（`danger` / `warning` / `info` / `success`）決定圖示和確認按鈕的顏色。

<ComponentPreview title="Confirm an action">
  <ConfirmModalDemo />
</ComponentPreview>

```tsx
"use client";

import { useState } from "react";
import { ConfirmModal, Button } from "@yuhuanowo/yunui";

export function ConfirmModalDemo() {
  const [open, setOpen] = useState(false);
  return (
    <>
      <Button variant="red" onClick={() => setOpen(true)}>Delete project</Button>
      <ConfirmModal
        isOpen={open}
        onClose={() => setOpen(false)}
        onConfirm={() => setOpen(false)}
        variant="danger"
        title="Delete project?"
        message="This permanently removes the project and all of its data. This action cannot be undone."
        confirmText="Delete"
      />
    </>
  );
}
```
