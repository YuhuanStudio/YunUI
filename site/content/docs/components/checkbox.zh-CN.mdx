import { Checkbox } from "yunui";

受控的复选框。它仅支持受控模式：传入 `checked` 和 `onCheckedChange`。将它与 `<label htmlFor>`（通过 `id` prop）配对，可构成一个无障碍的、点击即可切换的行。

## Import

```tsx
import { Checkbox } from "@yuhuanowo/yunui";
```

## Usage

`Checkbox` 是受控组件——将它与 state 关联：

```tsx
const [checked, setChecked] = useState(false);
<label className="flex items-center gap-2">
  <Checkbox checked={checked} onCheckedChange={setChecked} id="terms" />
  Accept terms
</label>
```
