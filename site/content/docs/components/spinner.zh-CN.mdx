import { Spinner } from "yunui";

一个带有内置、仅供屏幕阅读器朗读的「Loading…」标签的加载旋转图标。提供三种尺寸，并继承柔和的前景色，因此能自然地嵌入按钮、卡片和行内文字中。

## 导入

```tsx
import { Spinner } from "@yuhuanowo/yunui";
```
