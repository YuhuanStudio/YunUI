import { Accordion, AccordionItem, AccordionTrigger, AccordionContent } from "yunui";

一个基于 Radix、垂直堆叠的可折叠区块集合。设置 `type="single"` 可一次只展开一个面板（再加 `collapsible` 让已展开的面板可以收起），或设置 `type="multiple"` 允许同时展开多个。完整的键盘无障碍支持（方向键、Home/End、Enter/Space）来自 Radix。

## 导入

```tsx
import {
  Accordion, AccordionItem, AccordionTrigger, AccordionContent,
} from "@yuhuanowo/yunui";
```

## 属性

`Accordion`、`AccordionItem`、`AccordionTrigger` 和 `AccordionContent` 都是对 Radix Accordion 原语的轻量包装，因此接受它们的全部属性。最常用的几个：

<PropsTable
  rows={[
    { name: "type", type: '"single" | "multiple"', required: true, description: "一次可展开一个还是多个面板。" },
    { name: "collapsible", type: "boolean", defaultValue: "false", description: "（type=single）允许关闭已展开的面板。" },
    { name: "value / defaultValue", type: "string | string[]", description: "受控 / 非受控的展开项。" },
    { name: "onValueChange", type: "(value) => void", description: "展开项变化时触发。" },
    { name: "AccordionItem.value", type: "string", required: true, description: "该区块的唯一标识符。" },
    { name: "disabled", type: "boolean", description: "禁用单个项或整个组。" },
  ]}
/>
