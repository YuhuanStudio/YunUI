import { Tabs, TabsList, TabsTrigger, TabsContent } from "yunui";

一组基于 Radix 的选项卡面板。`Tabs` 管理当前激活的选项卡；通过 `TabsList`（触发器行）、`TabsTrigger`（单个可点击选项卡）和 `TabsContent`（对应选项卡激活时显示的面板）组合使用。完整的键盘无障碍支持（方向键、Home/End）由 Radix 提供。使用 `defaultValue` 实现非受控选项卡，或用 `value`/`onValueChange` 进行受控。

## 导入

```tsx
import {
  Tabs, TabsList, TabsTrigger, TabsContent,
} from "@yuhuanowo/yunui";
```

## 组成部分

| 部分 | 作用 |
| --- | --- |
| `Tabs` | 根组件；持有 `value`/`onValueChange` 或 `defaultValue`。 |
| `TabsList` | 触发器的横向容器。 |
| `TabsTrigger` | 可点击的选项卡（必填 `value`）；支持 `disabled`。 |
| `TabsContent` | 对应选项卡激活时显示的面板（必填 `value`）。 |

`Tabs` 以设计系统样式重新导出 Radix `Tabs` 的各个部分。完整的属性参考（含 `orientation` 与 `activationMode`）请见 [Radix Tabs 文档](https://www.radix-ui.com/primitives/docs/components/tabs)。
