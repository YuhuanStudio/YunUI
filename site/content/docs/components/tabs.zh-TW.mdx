import { Tabs, TabsList, TabsTrigger, TabsContent } from "yunui";

一組以 Radix 為基礎的頁籤面板。`Tabs` 管理目前作用中的頁籤；透過 `TabsList`（觸發器列）、`TabsTrigger`（單個可點擊頁籤）與 `TabsContent`（對應頁籤作用時顯示的面板）組合使用。完整的鍵盤無障礙支援（方向鍵、Home/End）由 Radix 提供。使用 `defaultValue` 實現非受控頁籤，或用 `value`/`onValueChange` 進行受控。

## 匯入

```tsx
import {
  Tabs, TabsList, TabsTrigger, TabsContent,
} from "@yuhuanowo/yunui";
```

## 組成部分

| 部分 | 作用 |
| --- | --- |
| `Tabs` | 根元件；持有 `value`/`onValueChange` 或 `defaultValue`。 |
| `TabsList` | 觸發器的橫向容器。 |
| `TabsTrigger` | 可點擊的頁籤（必填 `value`）；支援 `disabled`。 |
| `TabsContent` | 對應頁籤作用時顯示的面板（必填 `value`）。 |

`Tabs` 以設計系統樣式重新匯出 Radix `Tabs` 的各個部分。完整的屬性參考（含 `orientation` 與 `activationMode`）請見 [Radix Tabs 文件](https://www.radix-ui.com/primitives/docs/components/tabs)。
