import { Pagination } from "yunui";

一个受控的分页导航。它渲染上一页/下一页箭头以及带数字的页码按钮，并自动用省略号截断——第一页和最后一页始终可见，`siblingCount` 个相邻页围绕在当前页两侧。由宿主持有页码状态和路由：传入当前 `page` 和 `totalPages`，并在 `onPageChange` 时更新。当 `totalPages <= 1` 时不渲染任何内容。

## 导入

```tsx
import { Pagination } from "@yuhuanowo/yunui";
```

## 基础用法

`Pagination` 是受控的，因此将 `page`/`onPageChange` 接入你自己的 state（或路由）：

<ComponentPreview
  title="Pagination"
  code={`function Example() {
  const [page, setPage] = useState(3);
  return <Pagination page={page} totalPages={10} onPageChange={setPage} />;
}`}
>

</ComponentPreview>
