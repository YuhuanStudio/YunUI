import { Card, Badge, Button } from "yunui";

A surface container rendering the canonical `.card` style — the same look as the ~200 raw `className="card"` usages across Yunxin, in one component. Pass `hover` for an interactive lift.

## Import

```tsx
import { Card } from "@yuhuanowo/yunui";
```

## Basic

<ComponentPreview
  title="Card"

## Hover

`hover` adds a border + shadow transition for clickable cards.

<ComponentPreview
  title="Hover lift"
  code={`<Card hover className="p-5 w-60">
  <div className="flex items-center justify-between mb-2">
    <span className="font-medium">Pro plan</span>
    <Badge variant="success">Active</Badge>
  </div>
  <p className="text-caption">Hover me — the border and shadow animate.</p>
</Card>`}
>
  <Card hover className="p-5 w-60">
    <div className="flex items-center justify-between mb-2">
      <span className="font-medium">Pro plan</span>
      <Badge variant="success">Active</Badge>
    </div>
    <p className="text-caption">Hover me — the border and shadow animate.</p>
  </Card>
</ComponentPreview>

## Composing

`Card` is an unopinionated container — compose any content inside, including other primitives.

<ComponentPreview title="With action">
  <Card className="p-5 w-72">
    <p className="font-medium mb-1">Upgrade your plan</p>
    <p className="text-caption mb-4">Unlock higher rate limits and priority support.</p>
    <Button variant="primary" size="sm">Upgrade</Button>
  </Card>
</ComponentPreview>
