import { ShinyButton } from "yunui";

一个高强调的行动号召按钮，带有动画微光扫过效果和一个在悬停时向前移动的尾部箭头。用于页面上最重要的那一个操作。设置 `href` 可将其渲染为路由链接（通过已配置的 Link 适配器）；否则它会渲染为 `<button>` 并调用 `onClick`。

## 导入

```tsx
import { ShinyButton } from "@yuhuanowo/yunui";
```

## 作为链接

传入 `href` 即可渲染为路由链接而非按钮。

<ComponentPreview
  title="作为链接"
  code={`<ShinyButton href="/docs">Read the docs</ShinyButton>`}
>
  <ShinyButton href="/docs">Read the docs</ShinyButton>
</ComponentPreview>
