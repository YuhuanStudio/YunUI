import { AnimatedNumber } from "yunui";

一个在挂载时（以及每当 `value` 变化时）以弹簧动画从 0 增长到 `value` 的数字。它在服务端和客户端都从 0 开始，因此在 SSR 和水合期间渲染都是安全的。用 `suffix` 表示单位，用 `decimals` 表示小数精度。

## 导入

```tsx
import { AnimatedNumber } from "@yuhuanowo/yunui";
```

## 基本用法

<ComponentPreview
  title="Count up"
  code={`<AnimatedNumber value={1280} />`}
>
  <span className="text-4xl font-bold tabular-nums">
    <AnimatedNumber value={1280} />
  </span>
</ComponentPreview>

## 带后缀

用 `suffix` 属性追加单位，例如 `%` 或 `K`。

<ComponentPreview
  title="Suffix"
  code={`<AnimatedNumber value={98} suffix="%" />`}
>
  <span className="text-4xl font-bold tabular-nums">
    <AnimatedNumber value={98} suffix="%" />
  </span>
</ComponentPreview>

## 小数

为小数值设置 `decimals`；末尾的零会被去除。

<ComponentPreview
  title="Decimal places"
  code={`<AnimatedNumber value={4.8} decimals={1} />`}
>
  <span className="text-4xl font-bold tabular-nums">
    <AnimatedNumber value={4.8} decimals={1} />
  </span>
</ComponentPreview>
