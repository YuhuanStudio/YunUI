import { Breadcrumb, BreadcrumbList, BreadcrumbItem, BreadcrumbLink, BreadcrumbSeparator, BreadcrumbPage, BreadcrumbEllipsis } from "yunui";

一個無障礙的麵包屑導覽。`Breadcrumb` 是包裹有序 `BreadcrumbList` 的 `<nav>` 地標；每個 `BreadcrumbItem` 可容納 `BreadcrumbLink`（透過 YunUI 的 `Link` 轉接器路由）、目前的 `BreadcrumbPage` 或 `BreadcrumbSeparator`。使用 `BreadcrumbEllipsis` 折疊較長的路徑。

## 匯入

```tsx
import {
  Breadcrumb, BreadcrumbList, BreadcrumbItem, BreadcrumbLink,
  BreadcrumbSeparator, BreadcrumbPage, BreadcrumbEllipsis,
} from "@yuhuanowo/yunui";
```

## 組成部分

| 部分 | 作用 |
| --- | --- |
| `Breadcrumb` | 根 `<nav>` 地標（`aria-label` 預設為 `"Breadcrumb"`）。 |
| `BreadcrumbList` | 容納各項目的有序 `<ol>`。 |
| `BreadcrumbItem` | 路徑中的單個 `<li>`。 |
| `BreadcrumbLink` | 可導覽的連結；透過 YunUI 的 `Link` 轉接器路由（必填 `href`）。 |
| `BreadcrumbPage` | 目前頁面；不可互動並標記 `aria-current="page"`。 |
| `BreadcrumbSeparator` | 視覺分隔符（預設為箭頭）；對輔助技術隱藏。 |
| `BreadcrumbEllipsis` | 用於在較長路徑中折疊中間項目的佔位符。 |
