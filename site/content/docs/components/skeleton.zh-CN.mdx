import { Skeleton } from "yunui";

用于加载状态的脉冲占位块。它渲染一个柔和、圆角的 `<div>`，带有轻微的 `animate-pulse` 动画。完全通过 `className` 设定其尺寸与形状——为文字行设置宽高，或用 `rounded-full` 表示头像——并组合多个以勾勒出你正在等待的布局。

## 导入

```tsx
import { Skeleton } from "@yuhuanowo/yunui";
```

## 属性

`Skeleton` 就是一个普通的 `<div>`，因此接受所有标准 `<div>` 属性。它没有自定义属性——尺寸、形状和间距完全通过 `className` 控制。
