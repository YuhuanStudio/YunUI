import { Badge } from "yunui";

用于状态和标签的小型行内胶囊。它渲染语义化的颜色变体，并转发所有原生 `<span>` 属性。

## 导入

```tsx
import { Badge } from "@yuhuanowo/yunui";
```

## 变体

<ComponentPreview
  title="变体"
  code={`<Badge>Default</Badge>
<Badge variant="success">Success</Badge>

<Badge variant="error">Error</Badge>
<Badge variant="info">Info</Badge>`}

</ComponentPreview>

## 结构性变体

`secondary`、`outline` 与 `destructive` 补齐了面向 shadcn 风格调用点的变体（`destructive` 是 `error` 的别名）。

<ComponentPreview
  title="结构性变体"
  code={`<Badge variant="secondary">Secondary</Badge>
<Badge variant="outline">Outline</Badge>
<Badge variant="destructive">Destructive</Badge>`}
>
  <Badge variant="secondary">Secondary</Badge>
  <Badge variant="outline">Outline</Badge>
  <Badge variant="destructive">Destructive</Badge>
</ComponentPreview>
