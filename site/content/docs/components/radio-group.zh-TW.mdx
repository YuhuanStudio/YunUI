import { RadioGroup, RadioGroupItem } from "yunui";

一個基於 Radix 的單選項集合。透過 `value`/`onValueChange` 受控（或透過 `defaultValue` 非受控）。方向鍵可在組內移動並選擇，整個組是單個 tab 停靠點。每一項渲染一個圓環，選中時填充主強調色並顯示一個中心圓點——與 YunUI 的 Checkbox/Switch 視覺語言一致。

## 匯入

```tsx
import { RadioGroup, RadioGroupItem } from "@yuhuanowo/yunui";
```

## 屬性

`RadioGroup` 和 `RadioGroupItem` 包裝了 Radix 的 RadioGroup 原語，接受它們的全部屬性。最常用的幾個：
