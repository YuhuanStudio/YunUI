import { Label, Input } from "yunui";

帶樣式的表單標籤。它渲染一個原生 `<label>`，採用設計系統的排版樣式，因此可以透過 `htmlFor` 將其與任意控制項配對，實現點擊聚焦和螢幕閱讀器關聯。所有原生 `<label>` 屬性都會透傳。

## Import

```tsx
import { Label } from "@yuhuanowo/yunui";
```

## With a field

<ComponentPreview
  title="Label + Input"
  code={`<Label htmlFor="email">Email</Label>
<Input id="email" placeholder="you@example.com" />`}
>
  <div className="w-full max-w-xs space-y-1.5">
    <Label htmlFor="email">Email</Label>
    <Input id="email" placeholder="you@example.com" />
  </div>
</ComponentPreview>

`Label` 自身沒有任何必填的 props——它會轉發每一個原生 `<label>` 屬性。使用 `htmlFor` 指向控制項的 `id`。
