import { Slider } from "yunui";

基于 Radix `Slider` 构建的范围滑块。它渲染设计系统的轨道、范围和滑块，并转发每一个 Radix Slider 属性（`defaultValue`、`value`/`onValueChange`、`min`、`max`、`step`、`disabled`）。

## Import

```tsx
import { Slider } from "@yuhuanowo/yunui";
```
