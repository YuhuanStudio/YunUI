import { Progress } from "yunui";

基于 Radix Progress 构建的水平进度条。传入 0 到 100 之间的 `value`，已填充的指示条会动画过渡到该百分比。所有 Radix Progress 属性都会透传，`className` 可用于调整轨道的高度或颜色。

## 导入

```tsx
import { Progress } from "@yuhuanowo/yunui";
```

## 属性

`Progress` 包装了 Radix Progress 根元素，因此接受所有 Radix Progress 属性以及 `className`。最常设置的属性：
