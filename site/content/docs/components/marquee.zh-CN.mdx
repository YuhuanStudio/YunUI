import { Marquee, Badge } from "yunui";

一个无限滚动的行（或列），它会重复其子元素以填满一个无缝循环。它支持反向滚动、悬停暂停以及垂直方向。可通过 `className` 用 `--duration` 和 `--gap` CSS 变量调节速度和间距。

## 导入

```tsx
import { Marquee } from "@yuhuanowo/yunui";
```

## 基本用法

<ComponentPreview
  title="滚动行"
  code={`<Marquee>
  <Badge>React</Badge>
  <Badge>Next.js</Badge>
  <Badge>Tailwind</Badge>
  <Badge>Radix</Badge>
</Marquee>`}
>
  <div className="w-full">
    <Marquee>
      <Badge>React</Badge>
      <Badge>Next.js</Badge>
      <Badge>Tailwind</Badge>
      <Badge>Radix</Badge>
    </Marquee>
  </div>
</ComponentPreview>

## 反向与悬停暂停

设置 `reverse` 可反向滚动，设置 `pauseOnHover` 可在指针悬停其上时停止动画。

<ComponentPreview
  title="反向 + 悬停暂停"
  code={`<Marquee reverse pauseOnHover>
  <Badge variant="info">Vision</Badge>
  <Badge variant="success">Thinking</Badge>
  <Badge variant="warning">Tools</Badge>
  <Badge variant="error">Audio</Badge>
</Marquee>`}
>
  <div className="w-full">
    <Marquee reverse pauseOnHover>
      <Badge variant="info">Vision</Badge>
      <Badge variant="success">Thinking</Badge>
      <Badge variant="warning">Tools</Badge>
      <Badge variant="error">Audio</Badge>
    </Marquee>
  </div>
</ComponentPreview>

## 自定义速度

通过 `className` 覆盖 `--duration` 以减慢或加快循环。

<ComponentPreview
  title="更慢的循环"
  code={`<Marquee className="[--duration:20s]">
  <Badge>Alpha</Badge>
  <Badge>Beta</Badge>
  <Badge>Gamma</Badge>
</Marquee>`}
>
  <div className="w-full">
    <Marquee className="[--duration:20s]">
      <Badge>Alpha</Badge>
      <Badge>Beta</Badge>
      <Badge>Gamma</Badge>
    </Marquee>
  </div>
</ComponentPreview>
