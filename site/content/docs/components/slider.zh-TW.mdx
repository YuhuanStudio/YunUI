import { Slider } from "yunui";

基於 Radix `Slider` 建構的範圍滑桿。它渲染設計系統的軌道、範圍和滑桿，並轉發每一個 Radix Slider 屬性（`defaultValue`、`value`/`onValueChange`、`min`、`max`、`step`、`disabled`）。

## Import

```tsx
import { Slider } from "@yuhuanowo/yunui";
```

`Slider` 是 Radix `Slider` 之上的輕量樣式封裝。完整的屬性參考（受控的 `value`/`onValueChange`、`minStepsBetweenThumbs`、方向等）請參閱 [Radix Slider 文件](https://www.radix-ui.com/primitives/docs/components/slider)。
