import { FAQ } from "yunui/patterns";

一个手风琴式的问题列表，每次只展开一项。它接收一个简单的 `items` 数组（`{ question, answer }`）并自行管理展开状态，因此宿主应用只需提供（并翻译）内容即可。使用 `defaultOpenIndex` 选择初始展开的项，或传入 `null` 让其初始全部折叠。

## Import

```tsx
import { FAQ } from "@yuhuanowo/yunui/patterns";
```

## Basic

<ComponentPreview
  title="FAQ"
  code={`<FAQ
  items={[
    { question: "What is YunUI?", answer: "A prop-driven React component library." },
    { question: "Is it free?", answer: "Yes, it is open source under Apache-2.0." },
    { question: "Does it support i18n?", answer: "Yes, via injected adapters." },
  ]}
/>`}
>
  <div className="w-full max-w-2xl">
    <FAQ
      items={[
        { question: "What is YunUI?", answer: "A prop-driven React component library." },
        { question: "Is it free?", answer: "Yes, it is open source under Apache-2.0." },
        { question: "Does it support i18n?", answer: "Yes, via injected adapters." },
      ]}
    />
  </div>
</ComponentPreview>

## Start collapsed

传入 `defaultOpenIndex={null}`，让每一项初始都处于折叠状态。
