「活跃会话 / 已登录设备」列表中的一行：设备图标、带当前 / 失效徽章的设备名、`浏览器 · 系统` 详情行、IP + 最后活动时间页脚，以及一个吊销按钮。仅负责展示——宿主把设备类型映射到图标、格式化时间，并负责吊销操作。

## 导入

```tsx
import { SessionItem } from "@yuhuanowo/yunui/patterns";
```

## 示例

`current` 显示徽章并隐藏吊销按钮;`inactive` 让该行变暗。设备 `icon` 与相对 `time` 都是宿主提供的插槽。
