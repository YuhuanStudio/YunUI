import { Sidebar } from "yunui/patterns";

应用导航侧边栏：包含 logo、带激活状态匹配的分组导航 `sections`、移动端抽屉、桌面端折叠，以及底部 `footer` 插槽。路由通过注入的 Link 适配器流转，你也可以用 `onNavigate` 自行掌控导航。每个导航项都是纯数据（`label` + `href`）；在应用代码中可为每项添加可选的 `icon` 组件。

侧边栏使用了 `position: fixed`，因此预览将其放在一个带 `transform` 的框架舞台内（这会让固定定位的子元素相对于该框架而非视口进行定位）。

## 导入

```tsx
import { Sidebar } from "@yuhuanowo/yunui/patterns";
```
