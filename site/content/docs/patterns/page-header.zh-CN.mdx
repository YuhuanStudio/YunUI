import { PageHeader } from "yunui/patterns";
import { Button } from "yunui";

一致的页面标题区块：一个大号标题、可选的 `description`，以及固定在右侧的可选 `actions` 插槽（按钮、菜单等）。把它放在任意页面顶部，即可获得统一的间距与排版。

## 导入

```tsx
import { PageHeader } from "@yuhuanowo/yunui/patterns";
```

## 基础用法

<ComponentPreview
  title="PageHeader"
  code={`<PageHeader
  title="仪表盘"
  description="An overview of your account activity."
/>`}
>
  <div className="w-full max-w-xl">
    <PageHeader
      title="仪表盘"
      description="An overview of your account activity."
    />
  </div>
</ComponentPreview>

## 带操作按钮

`actions` 插槽可接收任意节点——通常是一个或多个按钮。

<ComponentPreview
  title="带操作"
  code={`<PageHeader
  title="项目"
  description="Manage your active projects."
  actions={<Button variant="primary">New project</Button>}
/>`}
>
  <div className="w-full max-w-xl">
    <PageHeader
      title="项目"
      description="Manage your active projects."
      actions={<Button variant="primary">New project</Button>}
    />
  </div>
</ComponentPreview>
