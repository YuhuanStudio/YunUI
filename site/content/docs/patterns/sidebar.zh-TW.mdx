import { Sidebar } from "yunui/patterns";

應用導覽側邊欄：包含 logo、帶啟用狀態匹配的分組導覽 `sections`、行動裝置抽屜、桌面端收合，以及底部 `footer` 插槽。路由透過注入的 Link 轉接器流轉，你也可以用 `onNavigate` 自行掌控導覽。每個導覽項都是純資料（`label` + `href`）；在應用程式碼中可為每項加入可選的 `icon` 元件。

側邊欄使用了 `position: fixed`，因此預覽將其放在一個帶 `transform` 的框架舞台內（這會讓固定定位的子元素相對於該框架而非視窗進行定位）。

## Import

```tsx
import { Sidebar } from "@yuhuanowo/yunui/patterns";
```
