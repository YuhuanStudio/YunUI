import { BlogCard } from "yunui/patterns";

博客文章预览卡片，包含封面图、分类徽章、元信息行（日期、阅读时长、作者）和标签。它通过注入的 `Link` 适配器链接到文章，并提供在 `md`+ 网格上横跨两列的 `featured` 变体。

## 引入

```tsx
import { BlogCard } from "@yuhuanowo/yunui/patterns";
```
