import { SessionItem } from "yunui/patterns";

「活躍工作階段 / 已登入裝置」清單中的一列：裝置圖示、帶目前 / 失效徽章的裝置名、`瀏覽器 · 系統` 詳情列、IP + 最後活動時間頁尾，以及一個撤銷按鈕。僅負責展示——宿主把裝置類型對應到圖示、格式化時間，並負責撤銷操作。

## 匯入

```tsx
import { SessionItem } from "@yuhuanowo/yunui/patterns";
```

## 範例

`current` 顯示徽章並隱藏撤銷按鈕;`inactive` 讓該列變暗。裝置 `icon` 與相對 `time` 都是宿主提供的插槽。
