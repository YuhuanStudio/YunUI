import { CalloutBlock } from "yunui/content";

對齊 YunUI `Alert` 元件樣式的提示框——整塊柔和色底、四邊柔和邊框,帶語義化圖示與標題,與 Alert、Badge、狀態點共用同一套紅/綠/琥珀色語彙。六種類型涵蓋 GitHub 的告示集合：`note`、`tip`、`important`、`warning`、`caution`，外加 YunUI 的 `success`。在 `MarkdownRenderer` 內，它們會由 `> [!NOTE]` 引用區塊語法自動生成；當你組合自己的版面時可直接使用 `CalloutBlock`。配套的 `parseCalloutType()` 輔助函式會偵測引用區塊首行的 `[!TYPE]` 標記。

## 匯入

```tsx
import { CalloutBlock, parseCalloutType } from "@yuhuanowo/yunui/content";
```

## 在 markdown 中

在 `MarkdownRenderer` 內，提示框來自標準的 GitHub 引用區塊語法：

```md
> [!WARNING]
> This action cannot be undone.
```
