一个受控的全屏图片查看器，通过 portal 渲染到 `document.body`。它支持缩放、旋转、下载与键盘快捷键（`Esc` 关闭、`+` / `-` 缩放、`R` 旋转）。它就是 [`ContentImage`](/docs/content/content-image) 点击时打开的查看器，但你也可以为任意图片自行驱动它：持有打开状态并传入 `isOpen` / `onClose`。

## 导入

```tsx
import { ImageLightbox } from "@yuhuanowo/yunui/content";
```

## 用法

```tsx
const [open, setOpen] = useState(false);

<button onClick={() => setOpen(true)}>View</button>
<ImageLightbox
  src={imageUrl}
  alt="Generated image"
  isOpen={open}
  onClose={() => setOpen(false)}
/>
```
