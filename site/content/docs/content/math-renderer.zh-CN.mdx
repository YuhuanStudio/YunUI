import { MathRenderer } from "yunui/content";

用 [KaTeX](https://katex.org) 渲染 LaTeX 表达式。把表达式作为 `math` 传入；设置 `block` 得到居中的展示式，或不设置以获得随上下文排版的行内公式。`InlineMath` 与 `BlockMath` 是接收 `value` 属性的便捷封装。在 `MarkdownRenderer` 内，`$…$` 与 `$$…$$` 会自动通过本组件渲染。

## 导入

```tsx
import { MathRenderer, InlineMath, BlockMath } from "@yuhuanowo/yunui/content";
```

需要宿主应用引入 `katex/dist/katex.min.css`。
