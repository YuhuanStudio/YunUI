import { IDBadge } from "yunui/ai";

用于 id、密钥及其他可复制令牌的紧凑等宽徽章。点击即可将 `text` 复制到剪贴板，并短暂显示对勾图标。过长的值默认会被截断；设置 `truncate={false}` 可在一行内保留完整值。

## Import

```tsx
import { IDBadge } from "@yuhuanowo/yunui/ai";
```

<ComponentPreview
  title="Basic"

</ComponentPreview>
