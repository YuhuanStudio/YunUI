import { Footer, GithubIcon, DiscordIcon, InstagramIcon } from "yunui/ai";

網站頁尾：包含圖示與標語的品牌區塊、連結欄（`FooterSection[]`）、版權行，以及底部列的社群圖示連結（`FooterSocial[]`）。連結透過設定的 `Link` 配接器進行。該套件還匯出零相依的 `GithubIcon`、`DiscordIcon` 與 `InstagramIcon` 供社群插槽使用。

## Import

```tsx
import { Footer, GithubIcon, DiscordIcon, InstagramIcon } from "@yuhuanowo/yunui/ai";
```
