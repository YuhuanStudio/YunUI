import { Footer, GithubIcon, DiscordIcon, InstagramIcon } from "yunui/ai";

站点页脚：包含徽标与标语的品牌区块、链接栏（`FooterSection[]`）、版权行，以及底部栏的社交图标链接（`FooterSocial[]`）。链接通过配置的 `Link` 适配器进行。该包还导出零依赖的 `GithubIcon`、`DiscordIcon` 和 `InstagramIcon` 供社交插槽使用。

## 导入

```tsx
import { Footer, GithubIcon, DiscordIcon, InstagramIcon } from "@yuhuanowo/yunui/ai";
```
