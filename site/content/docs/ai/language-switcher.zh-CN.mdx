import { LanguageSwitcher } from "yunui/ai";

语言切换下拉菜单。它是受控组件——由你提供可用的 `locales`、`currentLocale` 以及 `onChange` 处理函数。实际的语言切换（cookie、刷新或路由跳转）由宿主负责。点击外部即可关闭，并提供 `icon` 与 `pill` 两种变体。

## Import

```tsx
import { LanguageSwitcher } from "@yuhuanowo/yunui/ai";
```

## Basic

在 state 中保存当前语言；在真实应用中，`onChange` 会设置 cookie 并刷新页面。

<ComponentPreview
  title="Basic"
  code={`import { useState } from "react";
import { LanguageSwitcher } from "@yuhuanowo/yunui/ai";

function Example() {
  const [locale, setLocale] = useState("en");
  return (
    <LanguageSwitcher
      currentLocale={locale}
      onChange={setLocale}
      variant="pill"
      locales={[
        { value: "en", label: "English" },
        { value: "zh-CN", label: "简体中文" },
        { value: "zh-TW", label: "繁體中文" },
      ]}
    />
  );
}`}
>
  <LanguageSwitcherDemo />
</ComponentPreview>
