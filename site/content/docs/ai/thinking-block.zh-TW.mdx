import { ThinkingBlock } from "yunui/ai";

用於展示模型推理/「思考」過程的可折疊面板。它自行管理展開狀態，帶有展開/折疊動畫，並在 `isStreaming` 時顯示即時指示。內容預設以保留換行的純文字算繪；傳入 `renderContent` 可接入 Markdown 算繪器。

## 匯入

```tsx
import { ThinkingBlock } from "@yuhuanowo/yunui/ai";
```

## 基礎用法

<ComponentPreview
  title="基礎用法"

  content="The user is asking for the capital of France. That is Paris. I'll answer directly and concisely."

/>`}

</ComponentPreview>

## 串流輸出

當 `isStreaming` 為 true 時，標題會顯示活躍徽章，並有一個閃爍游標跟隨文字。

<ComponentPreview
  title="串流"
  code={`<ThinkingBlock
  content="Let me work through the steps one by one"
  isStreaming
  defaultOpen
/>`}
>
  <div className="w-full max-w-lg">
    <ThinkingBlock
      content="Let me work through the steps one by one"

      defaultOpen
    />
  </div>
</ComponentPreview>
