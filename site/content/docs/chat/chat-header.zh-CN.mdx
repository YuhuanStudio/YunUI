聊天头部外壳：一条半透明、带背景模糊的横条，含 `left`、`status` 与 `actions` 插槽。把标题、模型选择器（如 YunUI 的 `ModelSelect`）、模式胶囊或开关组合进插槽——或者传入 `children` 完全覆盖插槽布局。

## 导入

```tsx
import { ChatHeader } from "@yuhuanowo/yunui/chat";
```
